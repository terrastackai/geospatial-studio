    Additional Resources¶

    Explore these resources to deepen your understanding of IBM Geospatial Studio and related technologies.

    📚 Official Documentation¶

    Geospatial Studio Documentation¶

    • Geospatial Studio Docs - Complete platform documentation
    • Geospatial Studio Toolkit SDK - Python SDK reference and examples
    • GitHub - Geospatial Studio - Main platform repository
    • GitHub - Geospatial Studio Core - Core backend services
    • GitHub - Geospatial Studio UI - Frontend application
    • GitHub - Geospatial Studio Toolkit - Python SDK source

    🤖 Foundation Models & AI¶

    IBM Geospatial Foundation Models¶

    • Prithvi on Hugging Face - Pre-trained geospatial models
    • Prithvi Paper - Research paper on Prithvi foundation model
    • IBM Granite Models - IBM's family of foundation models

    Terratorch - Training Framework¶

    • Terratorch Documentation - Geospatial model training library
    • Terratorch GitHub - Source code and examples
    • Terratorch Paper - Research on geospatial model training

    🛰️ Geospatial Data Sources¶

    Satellite Data¶

    • NASA Harmonized Landsat Sentinel-2 (HLS) - Combined Landsat 8/9 and Sentinel-2 data
    • Sentinel Hub - Cloud-based satellite imagery platform
    • Copernicus Open Access Hub - Free Sentinel satellite data
    • USGS Earth Explorer - Landsat and other satellite data

    Geospatial Datasets¶

    • HLS Burn Scars Dataset - Training data for burn scar detection
    • HLS Flood Dataset - Flood mapping training data
    • GeoBench - Benchmark for geospatial models

    🔧 Technical Tools & Libraries¶

    Geospatial Processing¶

    • GDAL - Geospatial Data Abstraction Library
    • Rasterio - Python library for raster data
    • GeoPandas - Python library for vector data
    • QGIS - Open-source GIS desktop application

    Machine Learning¶

    • PyTorch - Deep learning framework
    • PyTorch Lightning - High-level PyTorch wrapper
    • MLflow - ML lifecycle management platform
    • Ray Tune - Hyperparameter optimization

    Visualization¶

    • GeoServer - Open-source geospatial server
    • Leaflet - JavaScript mapping library
    • Matplotlib - Python plotting library

    📦 Example Datasets¶

    Pre-computed Examples¶

    The workshop includes several pre-computed examples you can explore:

    • Flood Detection - Porto Alegre, Brazil (May 2024)
    • Burn Scar Mapping - Park Fire, California (August 2024)
    • Multi-temporal Analysis - Assam, India flooding (July 2024)

    These examples are available in the Studio UI and can be accessed through the SDK.
